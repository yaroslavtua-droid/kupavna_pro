# HERZEN / Старая Купавна — база знаний проекта

Этот репозиторий хранит единую структурированную базу знаний по проекту **«Старая Купавна»**.

## Главный файл

`knowledge/kupavna_master_knowledge_base.md`

Именно этот файл считается актуальной master-базой проекта.

## Источник истины

Фактические данные сверяются с сайтом:

https://kupavna.pro/

Если данные в базе противоречат сайту, приоритет имеет сайт.

## Как использовать

1. Открыть `knowledge/kupavna_master_knowledge_base.md`.
2. Искать нужный раздел: блоки, плюсы, технические характеристики, коммерческие условия, маркетинг, задачи.
3. При обновлении данных вносить правки через Pull Request.
4. В `changelog.md` фиксировать, что изменилось.

## Правило качества

Не добавлять неподтверждённые факты в основную базу. Гипотезы, старые материалы и черновики хранить отдельно.

## Структура

```text
knowledge/   — актуальная master-база
sources/     — исходные базы, выгрузки, материалы
exports/     — отдельные выборки из master-базы
archive/     — старые версии и архив
docs/       — инструкции по процессу и промпты
changelog.md — журнал изменений
CLAUDE.md    — инструкция для Claude
```

## Как загрузить в GitHub

1. Создать private repository.
2. Загрузить содержимое папки `kupavna_github_repo_v2`, не сам zip как один файл.
3. Проверить, что файл `knowledge/kupavna_master_knowledge_base.md` открывается в GitHub.
4. Подключить репозиторий в корпоративном Claude.
5. В Claude ссылаться на главный файл: `knowledge/kupavna_master_knowledge_base.md`.
