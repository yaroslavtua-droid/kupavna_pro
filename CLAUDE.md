# Инструкция для Claude

Ты работаешь с репозиторием базы знаний проекта **«Старая Купавна»**.

Главный файл:

`knowledge/kupavna_master_knowledge_base.md`

## Основные правила

1. Сайт https://kupavna.pro/ — главный источник истины по фактам.
2. Не выдумывай данные.
3. Не меняй факты без проверки источника.
4. Если информация не подтверждена, помечай её как `требует проверки`.
5. Не смешивай факты, гипотезы, маркетинговые формулировки и исторические материалы.
6. При обновлении базы сохраняй структуру master-файла.
7. Если находишь противоречие, создай блок `Конфликт данных`.
8. Все изменения кратко фиксируй в `changelog.md`.
9. Для рабочих задач создавай отдельные файлы в `exports/`, не ломая master-файл.
10. Пиши кратко, структурно, прикладно.

## Типовые задачи

- найти факт по проекту;
- выделить плюсы проекта;
- собрать список блоков;
- подготовить текст для рассылки;
- подготовить данные для отчёта;
- проверить противоречия;
- обновить master-базу;
- создать отдельную выборку из master-файла.

## Приоритет источников

1. https://kupavna.pro/
2. `knowledge/kupavna_master_knowledge_base.md`
3. документы в `sources/`
4. рабочие материалы и старые версии
5. гипотезы и черновики

При конфликте с сайтом оставляй версию сайта и фиксируй конфликт.
