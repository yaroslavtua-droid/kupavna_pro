# Sources

Здесь можно хранить исходные материалы:

- базы из личного GPT;
- базы из корпоративного GPT;
- выгрузки из CRM;
- отчёты рекламных каналов;
- PDF;
- технические документы;
- старые версии материалов.

Материалы из этой папки не считаются фактами автоматически. Факты должны быть проверены и внесены в master-файл.
